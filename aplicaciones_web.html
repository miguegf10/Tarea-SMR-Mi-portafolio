Aplicaciones Web

Este módulo trata sobre el desarrollo, instalación y administración de aplicaciones web, utilizando tecnologías como HTML5, CSS3, JavaScript, PHP y bases de datos como MySQL. También se abordan herramientas de publicación como GitHub Pages y plataformas como XAMPP y servidores web reales.

🛠️ Herramientas y tecnologías utilizadas

  • HTML5 y CSS3 para el diseño y maquetación de páginas web.
  • Bootstrap 5 para diseño responsivo y moderno.
  • JavaScript para mejorar la interactividad del usuario.
  • PHP y MySQL para aplicaciones web dinámicas.
  • Git y GitHub para control de versiones y despliegue web.

├── css/

├── js/

├── img/

└── contacto.html

💡 Conceptos importantes aprendidos

  • Uso de formularios HTML para envío de datos.
  • Gestión de bases de datos con phpMyAdmin.
  • Creación de sitios web estáticos y dinámicos.
  • Publicación del portafolio profesional en GitHub Pages.
