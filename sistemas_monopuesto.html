Sistemas Operativos Monopuesto

Este módulo se centra en la instalación, configuración y mantenimiento de sistemas operativos en entornos monopuesto, es decir, en equipos individuales. Aprendemos a manejar sistemas operativos como Windows y Linux.

📘 Contenidos destacados

  • Instalación de sistemas operativos.
  • Configuración de usuarios y permisos.
  • Gestión de recursos del sistema.
  • Solución de problemas comunes.

💻 Herramientas utilizadas

Instaladores de sistemas operativos, herramientas de configuración, utilidades de diagnóstico de hardware.

📷 Evidencias

Proceso de instalación de un sistema operativo Windows.

Configuración de opciones de sistema y usuarios en Linux.
